# EffMind: AI-assisted Mind Mapping Tool

EffMind is an open-source mind map editor that runs on both Web and Visual Studio Code. Approximately 70% of the code and this documentation are assisted by AI tools such as **GitHub Copilot, ChatGPT, and Google Gemini**.

> ‚ö†Ô∏è Note: The code was generated by different AI tools, so coding style may vary. Functionality is complete, but code structure has not been fully organized yet.

## Tech Stack

* Components built with **Lit.js** ([https://lit.dev/](https://lit.dev/))
* Native Web Components
* **SVG** rendering for nodes and connections
* SPA architecture; VSCode extension reuses the same front-end via Webview
* Built-in multilingual system (JSON dictionaries + runtime switching)
* **Supports dark/light themes, automatically matches system preference**

## Features

### Mind Map Editing

* Create, edit, delete, and move nodes
* Manipulate child and sibling nodes
* Pan and zoom canvas
* Automatic layout
* Undo / redo
* Mini-map
* Node search
* Node and canvas right-click menus

### Import / Export

* Import: `.mind` (JSON)
* Export: `.mind`, `.png`, interactive collapsible `.svg`

### Multilingual Support

* Text maintained in JSON
* Runtime language switching
* Shared language configuration for Web & VSCode
* Extendable with custom language packs

### Themes

* Dark and light themes
* Follows system `prefers-color-scheme` by default

## Platform Support

### PWA

* Installable on desktop or mobile
* Offline support
* Tested on iOS 16 / iPadOS 16
* Android testing pending
